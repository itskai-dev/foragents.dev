# ⚡ forAgents.dev Favicon

## Design
- **Theme:** Lightning bolt - minimal, technical, agent-focused
- **Colors:** Amber/gold (#fbbf24) on dark background (#0a0a0a)
- **Style:** Sharp, geometric lightning bolt that works at 16px

## Files Created
✓ `favicon.ico` - Multi-size (16x16 + 32x32) Windows icon
✓ `favicon-16x16.png` - 16×16 PNG
✓ `favicon-32x32.png` - 32×32 PNG  
✓ `apple-touch-icon.png` - 180×180 PNG for iOS/Apple devices
✓ `favicon.svg` - Source vector file

## Preview
Open `favicon-preview.html` in a browser to see all sizes side-by-side.

## HTML Integration
Add to `<head>`:
```html
<link rel="icon" type="image/x-icon" href="/favicon.ico">
<link rel="icon" type="image/png" sizes="32x32" href="/favicon-32x32.png">
<link rel="icon" type="image/png" sizes="16x16" href="/favicon-16x16.png">
<link rel="apple-touch-icon" sizes="180x180" href="/apple-touch-icon.png">
```
